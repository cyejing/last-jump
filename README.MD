# Last Jump 

for happy surfing in internet

## Start 
1. install cia in outer net

2. install spy for socks5 proxy

3. config socks5 proxy in browser 

4. enjoy

## Config 
1. ``--port`` 配置端口
2. ``--remoteHost`` 配置cia远程地址
3. ``--remotePort`` 配置cia远程端口
4. ``--cryptoName`` 加密算法
5. ``--cryptoPassword`` 加密密文
6. ``--auth`` socks5密文
